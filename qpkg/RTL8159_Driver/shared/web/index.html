RTL8159 USB Ethernet Driver for QNAP

Version: 2.20.1
Kernel: 5.10.60 (x86_64)
Architecture: x86_64

Description

High-performance kernel driver for Realtek RTL8152/RTL8153/RTL8156/RTL8157/RTL8159 USB Ethernet adapters. This driver enables support for high-speed USB network adapters up to 10Gbps on QNAP NAS systems.

Features

  • ✓ Supports up to 10Gbps USB 3.2 Gen 2x2 network adapters
  • ✓ Compatible with 2.5G/5G/10G USB Ethernet dongles
  • ✓ Optimized for QNAP QuTS hero h5.x.x and QTS 4.x/5.x
  • ✓ No system reboot required - driver loads immediately after installation
  • ✓ Automatic loading at system boot
  • ✓ Hot-plug support - connect adapter anytime

Supported Devices

  • RTL8152B/8153: Gigabit (1Gbps) USB Ethernet
  • RTL8156/8156B: 2.5 Gigabit USB Ethernet
  • RTL8157: 5 Gigabit USB Ethernet
  • RTL8159: 10 Gigabit USB Ethernet
Note: The driver is loaded automatically after installation. Simply connect your Realtek USB Ethernet adapter and it will be detected.

Verification

To verify the driver is loaded and working, connect via SSH and run:

# Check if driver is loaded
lsmod | grep r8152

# View driver information
modinfo r8152

# List network interfaces
ip link show
